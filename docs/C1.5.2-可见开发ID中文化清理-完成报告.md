## 四、修改方案

在两个文件内各添加一个极小的私有展示辅助函数，只做 `hero_001 → 剑无极` 的 UI 文案映射，不改内部 ID、不改存档、不改逻辑。

```typescript
/**
 * [C1.5.2] 将内部 heroId 映射为玩家可见的中文角色名。
 * 仅用于 UI 文案展示，不参与存档、逻辑或装备归属判断。
 */
private formatHeroDisplayName(heroId: string): string {
  if (heroId === 'hero_001') {
    return '剑无极';
  }
  return heroId;
}
```
