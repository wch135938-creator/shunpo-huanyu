## 钻石经济原则

钻石属于稀缺通用资源。

主要来源：

- 任务
- 活动
- 首次通关
- 自愿观看激励视频广告

主要用途：

- 后期高级装备成长
- 后期人物成长
- 法器、宠物、坐骑等高级成长

原则：

- 产出量必须少于金币与普通养成材料。
- 新手期基础成长不得强制消耗钻石。
- 不看广告仍可通过任务、活动和正常游玩获得少量钻石。

# 成长断层预警

出现以下情况必须调整：

连续2天战力无提升

连续3天无新角色

连续3天无新装备

连续2天无成长目标

---

# AI开发规则

新增任何系统时：

必须回答：

1. 是否增加成长感？
2. 是否增加长期目标？
3. 是否增加留存？
4. 是否影响成长曲线？

若答案全部为否：

禁止开发。
