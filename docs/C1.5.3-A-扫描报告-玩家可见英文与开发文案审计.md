| 属性 | 值 |
|---|---|
| **文件** | `assets/scripts/ui/EquipmentItemView.ts` |
| **行号** | 177–179 |

| **当前文案** | `HP+${vm.baseHp}` / `ATK+${vm.baseAtk}` / `DEF+${vm.baseDef}` |
| **玩家可见** | ✅ 是 — 装备背包列表中每个道具卡片的属性简览行 |
| **本地化状态** | **无任何覆盖**。EquipmentMediator 的 `_localizeEquipmentSummaryLabels()` 仅处理 `EquipmentPanel` 的 `hpBonusLabel/atkBonusLabel/defBonusLabel`，不触及 `EquipmentItemView.statsLabel`。 |

**代码片段：**
```typescript
// EquipmentItemView.ts:175-180
if (this.statsLabel) {
  const parts: string[] = [];
  if (vm.baseHp > 0) parts.push(`HP+${vm.baseHp}`);
  if (vm.baseAtk > 0) parts.push(`ATK+${vm.baseAtk}`);
  if (vm.baseDef > 0) parts.push(`DEF+${vm.baseDef}`);
  this.statsLabel.string = parts.join('  ');
}
```

**影响面：** 装备背包列表中所有道具卡片的属性行显示英文字母缩写，玩家可见。

**影响面：** 若未来 `_onPresenterRefresh()` 调用顺序变更，英文缩写可能重新暴露。
