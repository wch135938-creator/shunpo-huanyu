# AI开发规则

新增功能时必须说明：

1. 影响哪些KPI
2. 提升哪些数据
3. 风险是什么
4. 如何验证效果

无法回答以上问题：

禁止开发。

---

# V1.0成功标准

同时满足：

D1留存 ≥ 40%

D7留存 ≥ 18%

D30留存 ≥ 6%

广告观看率 ≥ 40%

人均广告观看 ≥ 6次

平均在线时长 ≥ 15分钟

---

达到以上指标：

视为V1.0成功。

进入内容扩展阶段。

---

# 最终原则

任何功能、

任何活动、

任何广告位、

任何数值调整，

最终都必须回答一个问题：

它是否让核心KPI变得更好？

如果没有，

则不应投入开发资源。
