# UI / UX设计规范

## 文件定位

本文件定义：

* UI设计规范
* UX设计规范
* 页面架构
* 导航架构
* 信息层级
* 新手引导
* 红点系统
* 弹窗系统

所有UI开发必须遵循本文件。

---

# 设计目标

UI承担：

1. 引导玩家
2. 告知目标
3. 提供反馈
4. 驱动留存
5. 驱动广告观看

---

# 核心原则

优先级：

↓

反馈

↓

效率

↓

美观

↓

炫酷

---

# 微信小游戏原则

玩家进入游戏后：

3秒内知道该做什么

10秒内完成首次点击

30秒内获得奖励

---

禁止：

思考超过5秒

找按钮超过3秒

---

# UI设计理念

采用：

少按钮

大按钮

强反馈

低学习成本

---

避免：

复杂菜单

多层嵌套

信息过载

---
